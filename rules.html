CHALLEXA OFFICIAL DISPUTE & REFEREE PROTOCOL
(Simple, firm, non-negotiable)

Your golden principle:

Players decide points.

Referee decides conflicts.

TYPES OF DISPUTES

We only recognize 3 official dispute categories

A — Score Disagreement

Example: “We think it’s 4-3” / “No it’s 3-3”

B — Line / Double Bounce Dispute

Example: Ball in/out, second bounce

C — Conduct Issue

Example: Coaching abuse, shouting, intimidation

Anything else = not valid dispute.

PLAYER PROCEDURE (MANDATORY)

Players must follow this order:

Stop play immediately

Raise hand

Call Tournament Controller

If you continue playing → dispute void.

CONTROLLER DECISION TREE

Controller ask ONE question only:

“What was the last point both teams agree on?”

CASE A — SCORE DISPUTE

Rule:

Return to last agreed score.

No replay of previous games.

CASE B — LINE / BOUNCE

Rule:

Point replayed.

No debate about who was right.

CASE C — CONDUCT

Controller gives:

Warning → Game Penalty → Match Default

No discussion.
